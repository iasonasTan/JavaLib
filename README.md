# Java Lib

WARNING:
This project is being moved into [JeLib](https://github.com/iasonasTan/JeLib) and this repo is now READONLY.

JavaLib is a library for java applications.

It includes pre-made classes for games, gui, xml, file handling etc.

To use it you can just download Lib-vX-X-X.jar and include it in your project as library,
or you can copy and paste the source code in your project (100% not recommended)
